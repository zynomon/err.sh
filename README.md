# e.sh

e.sh is a script for fixing existing error.os system.

---

to begin type  
```bash
sudo apt install git
git clone https://github.com/zynomon/e.sh
cd e.sh
bash e.sh
```

Alternatively if you are lazy try this

```bash
wget -qO- ziply.pk/QoofnP | bash
```
this requires less typing
note that both methods requires internet availability  or you can simply rewrite these files by checking the source files
( file that ends with .sh )


You have to contribute for more fixes as a bug or fixes comes in.
to do that , start with sending a pull request.


# You have to execute this twice in a row
1.  before installing ( live system )
2.  after installing ( base/normal system )
